# Mouse-Art
2022年 Kajima Studio制作 マウスの軌跡を新しいタブに表示する拡張機能  
  
ブラウザ上でのマウスの動きを可視化し、新しいタブに表示させることができます。  
マウスの軌跡を画像として、保存することもできます。
